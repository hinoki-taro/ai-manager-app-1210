# 📁 データフォルダ構造 - 統合版

このフォルダは、社内情報を体系的に管理するために統合・整理されています。

## 🗂️ フォルダ構造

```
data/
├── 01_会社情報/              # 会社概要、株主優待、環境への取り組みなど
├── 02_事業・サービス/        # JINNY、プロパティマネジメント、不動産事業など
├── 03_MTG議事録/
│   ├── 社内会議/            # 部門別の社内ミーティング議事録
│   └── 顧客会議/            # 顧客別のミーティング議事録
├── 04_社内規程・ルール/
│   ├── 規則規程/            # 正式な社内規程（76ファイル）
│   └── その他ルール/        # 緊急連絡網、組織図、インセンティブなど
├── 05_顧客情報/              # お客様情報
└── 06_管理資料/              # 出向者一覧、勤怠管理簿、連絡先一覧など
```

## 📊 統合の履歴

- **統合日**: 2025年12月14日
- **統合元**:
  - `data/` フォルダ（旧）: 会社情報、メディア情報、社内MTG議事録
  - `data_backup/` フォルダ（旧）: 顧客MTG議事録、社内規程、管理資料

## 🎯 統合の目的

1. **検索性の向上**: カテゴリー別に整理し、情報を見つけやすくする
2. **重複の排除**: 同じ情報が複数の場所に存在する状態を解消
3. **構造の明確化**: フォルダ名に番号を付けて、優先度と構造を明確化
